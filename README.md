# Sosyal Medya Paneli

Bu proje, sosyal medya platformları için bir yönetim paneli uygulamasıdır. React ve Vite kullanılarak geliştirilmiştir.


## Kurulum

```bash
npm install
```
```bash
npm run dev
```
```bash
npm run build
```

## Geliştirme Ortamı

- React
- Vite
- ESLint
- Tailwind CSS
